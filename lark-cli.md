飞书 CLI 提供了一系列登录、登出和查询登录状态的命令，具体操作如下：

## 相关操作命令

### 1. 以用户身份登录（授权）

执行该命令后，会打开授权链接，你需要在飞书中完成用户授权：

```
lark-cli auth login
```

完成授权后，AI 才可以访问你个人的日历、消息、文档等个人数据，以你的名义执行操作。如果暂时不授权，AI 仍然可以执行部分公开操作，但无法访问你的个人数据。

### 2. 查看当前登录状态

查询当前是否已经完成用户登录授权：

```
lark-cli auth status
```

### 3. 检查当前已有权限

查看当前应用/用户已获得的权限列表：

```
lark-cli auth check
```

### 4. 为指定业务域申请权限

如果你需要为某个业务域补充授权，可以执行：

```
lark-cli auth login --domain <domain>
```

将 `<domain>` 替换为你需要申请权限的业务域名称。

### 5. 登出

退出当前用户的登录授权：

```
lark-cli auth logout
```

### 6. 查看所有命令总览

如需查询所有可用命令，可以执行：

```
lark-cli help
```

如需查询某个具体命令的详细用法，可以执行：

```
lark-cli <command> --help
```
