• 这句话不是从某个“飞书机器人内置提示词”里直接取出来的，也不是数据库里配置的一段原文。

它大概率是模型根据运行上下文自己拼出来的：

- 我是当前飞书私聊里的机器人智能体
  来自飞书入口自动拼接的上下文：当前消息来源是飞书私聊/群聊，系统要求“只生成需要回到飞书的最终回复”。位置在 web-
  admin/api/services/feishu/feishu_bot_service.py:3578。

- 运行在 Hermes Agent / Codex 环境里
  这是污染/泄露式推断，不是应该给用户看的答案。来源混合了几类内部上下文：
  - 当前飞书连接器配置是 external_agent_type=hermes。
  - 外部 Agent 桥接会把 agent_type 传给 Runner，位置在 web-admin/api/services/chat/
    project_chat_execution_service.py:196。

  - 桌面端项目 Runner prompt 会写入“外部 Agent：Hermes”，位置在 web-admin/frontend/src/views/projects/
    ProjectChat.vue:6854。

  - 当前开发/排查环境本身是 Codex，所以模型把内部执行身份也混进去了。

结论：它不是“获取”来的权威身份，而是模型读到内部执行上下文后自己组织出来的错误用户可见表述。

正确回答应该类似：

> 我是当前飞书里的项目机器人/AI 助手，可以帮你处理项目消息、查询项目上下文、整理记录和协助操作飞书资源。
