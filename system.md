适合迁移到 /Volumes/work_mac_1_5T 的

- pnpm store：适合迁移，类似 npm cache。
- ~/.gradle：适合迁移，但建议用软链接或 GRADLE_USER_HOME。
- ~/.m2/repository：适合迁移，Maven 本地仓库。
- ~/.cache/uv：适合迁移，uv 缓存。
- pip cache：可以迁移，但你这里只有 9.5M，没必要。
- ~/.pyenv：这是 Python 解释器安装目录，不建议当缓存随便删；可以迁移，但风险比缓存高一点。

优先级建议

1. 先迁移 pnpm：6.4G
2. 再迁移 Gradle：7.3G
3. 再迁移 Maven：3.5G
4. 再迁移 uv：3.7G

这几项加起来大概能从本机盘挪走 20G 左右。
